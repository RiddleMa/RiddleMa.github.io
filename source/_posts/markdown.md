# 记录
## 1. 页面内链接

- 两种方法，要注意的点：
  - 采用小写字母
  - 使用连字符分隔单词，不要用空格和特殊字符
  - 保持命名唯一性
  - markdown语法有些平台不能解析

### 1.1 markdown语法
```markdown
<!-- 定义锚点 -->
<a id="section1"></a>
## 第一部分内容

[跳转到第二部分](#section2)

<!-- 目标锚点 -->
<a id="section2"></a>
## 第二部分内容

[返回顶部](#section1)
```
### 1.2 html语法
```markdown
## 目录
- [第一章](#chapter-1)
- [第二章](#chapter-2)

<a id="chapter-1"></a>
## 第一章
内容...

<a id="chapter-2"></a>
## 第二章
内容...

[返回第一章](#chapter-1)
```
- 目前这两种方法都只能给目录索引，如果在非目录位置使用链接，github上出错。

## 引用图片
```markdown
![替代文本](图片URL "可选标题")
```

## 折叠块
```markdown
<details>
<summary>点击展开详细说明</summary>

这里是折叠的详细内容，支持 Markdown 语法。

</details>
```

## 表格
```markdown

| 层级 | 符号 | 用途 |
|------|------|------|
| 1 级 | #    | 大章 |
| 2 级 | ##   | 小节 |

```
